## ⚙️ Usage

1. Clone repository `git clone git@github.com:hoangsvit/laravel-10x-sail-vercel.git`.
2. Browse single example, `cd laravel-10x-sail-vercel`.
3. Deploy to Vercel, `vercel`.

## 🚀 Click & Go
